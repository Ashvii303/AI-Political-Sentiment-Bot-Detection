## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/Ashvii303/AI-Political-Sentiment-Bot-Detection.git
```

### Backend

```bash
cd backend
pip install -r requirements.txt
python app.py
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
